# 파티원괌 REST API

Base URL Endpoint : `http://<DOMAIN>/api`

## Authorization & Headers

-   Token prefix 는 PG (파티원괌)
-   기본 `Content-Type` 는 `application/json`
-   인증이 필요한 API 의 경우는 `Authorization` 을 추가.

```json
{
    "Content-Type": "application/json",
    "Authorization": "PG <TOKEN>"
}
```
